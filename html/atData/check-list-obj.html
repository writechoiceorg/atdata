status string
The status of the list. See the table below for possible values.
Possible Values:

  • Pending: The list is currently being processed.
  • Results: The list has been processed, and results are available.
  • Completed: The list has been processed.
  • Processed: The list has been processed.
  • Failed: The processing of the list has failed.

records integer
The quantity of non-blank records in the file that have data to be processed. Header records will not be included in this count.
Example: 1002

size_bytes integer
The number of bytes of data that was originally contained in the list.
Example: 22709

created_at string
The date and time the list was submitted in ISO 8601 format. At all times, use the UTC zone.
Example: 2017-10-19T20:46:58Z"
